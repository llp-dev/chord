## embedded_hal::delay::DelayNs

*Trait*

Delay with up to nanosecond precision.

**Methods:**

- `delay_ns`: Pauses execution for at minimum `ns` nanoseconds. Pause can be longer
- `delay_us`: Pauses execution for at minimum `us` microseconds. Pause can be longer
- `delay_ms`: Pauses execution for at minimum `ms` milliseconds. Pause can be longer
