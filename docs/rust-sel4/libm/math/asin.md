## libm::math::asin::asin

*Function*

Arcsine (f64)

Computes the inverse sine (arc sine) of the argument `x`.
Arguments to asin must be in the range -1 to 1.
Returns values in radians, in the range of -pi/2 to pi/2.

```rust
fn asin(x: f64) -> f64
```
